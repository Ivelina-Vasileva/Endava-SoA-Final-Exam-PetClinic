🐾 Endava .Net SoA 2025 - Final Exam Project

📖 Overview
This repository contains the complete set of deliverables for the **Endava .Net School of Automation Final Exam**.
The project focuses on the **Spring PetClinic** application, covering the full testing lifecycle from requirements analysis to automated regression suites.

📁 Repository Structure
* **`/Documentation`**: Contains the original project requirements, environment setup guides, and exam tasks.
* **`/Manual`**: Includes the **Test Case Suite** and **Bug Reports** (Excel), documenting manual validation of the application's business logic.
* **`/SoA-Selenium-Framework-ExamTemplate`**: The core automation solution, featuring:
  * **UI Automation** (Selenium & Reqnroll)
  * **API Automation** (RestSharp)
  * **Unified Allure Reporting**
 
*For detailed instructions on how to run the automated tests, please refer to the README inside the **`/SoA-Selenium-Framework-ExamTemplate`** folder.*
